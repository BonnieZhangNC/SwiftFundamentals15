# CSC 118 – Swift Fundamentals I

**Spring 2026 A**

## Course Overview

This repository supports instruction for **CSC 118 – Swift Fundamentals I** at **Durham Technical Community College**. It contains course materials, instructional examples, sample projects, and reference code used to demonstrate foundational concepts in Swift programming and iOS application development.

## Course Information

* **Course:** CSC 118 – Swift Fundamentals I
* **Semester:** Spring 2026 A
* **Institution:** Durham Technical Community College

## Course Description

CSC 118 introduces the development of iOS and Apple applications using the Swift programming language. Instruction emphasizes Swift syntax, object-oriented programming principles, memory management, and functional programming concepts. Upon completion, students will be able to design and develop fully functional iOS and Apple applications using Swift.

## Textbook and Resources

* **Primary Textbook:** *Develop in Swift Fundamentals*
* **Development Environment:** Xcode 15

## Course Units

* **Unit 1:** Getting Started with App Development
* **Unit 2:** Introduction to UIKit
* **Unit 3:** Navigation and Workflows
